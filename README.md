🪄 Invisibility Cloak using OpenCV

An AI-inspired computer vision project built using Python and OpenCV that creates a real-time invisibility cloak effect similar to the one seen in Harry Potter movies.
The system detects a specific color (red cloth) using image processing and replaces it with the captured background, creating the illusion of invisibility through webcam-based live video processing.



🌟 Features

* 🎥 **Real-Time Webcam Processing**
  Captures and processes live video feed instantly.

* 🟥 **Color Detection using HSV**
  Detects red-colored objects/cloak accurately.

* 🪄 **Invisibility Effect**
  Replaces the detected cloak area with the background frame.

* ⚡ **Real-Time Computer Vision**
  Smooth and optimized live frame processing.

* 🧠 **Image Processing Techniques**
  Uses masking, dilation, morphology, and Gaussian blur.

* 🎨 **Interactive Visual Output**
  Live invisibility effect displayed using OpenCV windows.

⚙️ Working Principle

1. Webcam captures the background frame
2. System continuously reads live video frames
3. Red-colored cloth is detected using HSV color masking
4. Masked area is replaced with stored background
5. Creates an invisibility illusion in real time
